🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

ISSUE: Styles not applied
SOLUTION: 
  1. Hard refresh: Ctrl+Shift+R
  2. Clear cache: Ctrl+Shift+Delete
  3. Restart frontend server: npm start

ISSUE: Suggestions not loading
SOLUTION:
  1. Ensure backend is running: npm run dev (in backend folder)
  2. Check database connection
  3. Open DevTools (F12) → Network → Check API calls

ISSUE: FAQ accordion not working
SOLUTION:
  1. Check browser console for errors (F12)
  2. Ensure React is loaded properly
  3. Restart frontend server

ISSUE: Layout broken on mobile
SOLUTION:
  1. Clear browser cache
  2. Hard refresh the page
  3. Check viewport meta tag in index.html
  4. Use DevTools responsive design mode
